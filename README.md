## 👋 About Me

**Software Engineer at Diversified Automation** | **M.S. Computer Science, Indiana University Bloomington**

I build production-grade scalable systems and robust architectures — from real-time data logging and complex PLC logic to sophisticated e-banking backends and highly available web platforms. I thrive on solving complex engineering challenges and optimizing application performance across the entire stack.

**What I do day-to-day:**
- 🔍 Architect object-oriented backend solutions to streamline workflows and reduce overhead
- 📊 Optimize high-throughput, real-time data logging pipelines
- ⚙️ Develop full-stack interfaces and RESTful APIs using modern frameworks
- ⚡ Ship resilient platforms with CI/CD, containerization, and cloud infrastructure

**Previously:** Full Stack Engineer Intern at Twin Cities Innovation Alliance · Software Developer Intern at DVN IT Solutions · Software Engineer Intern at Prosoft Informatics
